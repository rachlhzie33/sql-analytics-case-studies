# Findings

## 1. Highest-volume ticket categories
Authentication, Integrations, and Workflow Automation issues generated the highest ticket volume in the sample dataset. This suggests that login access and technical integrations are common friction points for users.

## 2. Longest average resolution times
Integration-related issues such as ATS connection failures and webhook timeouts had the longest average resolution times, indicating higher technical complexity and possible dependency on engineering or third-party systems.

## 3. SLA breach drivers
Urgent API Errors and complex integration tickets showed the highest SLA breach rates. This suggests that current SLA targets may be too aggressive for technically complex issue types.

## 4. Channel-based patterns
Chat tickets generally received faster first responses, while email-based tickets took longer to resolve end-to-end. This may reflect that more complex cases are submitted via email.

## 5. Customer segment insights
Enterprise customers had fewer tickets overall but their issues were more likely to involve integrations and workflow automation, increasing average resolution time.

## 6. Recommended next steps
- Improve help center documentation for authentication issues
- Review routing logic for integration-related tickets
- Introduce automation for repetitive low-complexity cases
- Reassess SLA targets for complex technical tickets
